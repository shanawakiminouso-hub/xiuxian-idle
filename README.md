# 放置修仙

> 一人、一蒲团、一卷残经——自炼气起步，历十重境界，渡九重天劫，霞举飞升。
> 纯前端放置修仙网页游戏：无需联网、无需安装、无需构建，**双击 `index.html` 即玩**（Chrome / Edge，手机与 PC 浏览器皆可）。存档保存在你自己浏览器的本地存储里，不上传、不联网。

## 如何开始

1. 双击打开 `index.html`，即入仙途，自动生成你的散修角色与 30~50 位道友。
2. 挂机即修炼：修为每秒自增，攒满便点「突破」。
3. 想换设备？在**设置页**导出存档字符串，到新设备粘贴导入即可续缘。

## 三十秒上手

- **核心循环**：挂机攒修为 → 突破升境界 → 解锁新系统 → 效率更高地攒修为。
- **开局跟着「修行录」走**：十个新手目标引路（吐纳、功法、灵宠、论道……），完成即领灵玉灵石灵宠蛋。
- 修炼页中央的小人会随你的修行呼吸吐纳——**点他一下，手动吐纳立得约 20 秒修为**；突破、顿悟、闭关、奇遇，他都看在眼里。
- **每天上线三分钟**：一键领取（离线收益）→ 一键修炼（切打坐/闭关）→ 一键派遣（灵宠历练）→ 一键扫荡（镇妖塔）→ 顺道与道友论道一圈。
- 突破大境界有风险，失败莫慌：成功率保底逐次 +8%，备上破境丹更稳。

## 境界线

炼气 → 筑基 → 金丹 → 元婴 → 化神 → 炼虚 → 合体 → 大乘 → 渡劫 → 飞升，**每境十层**。
首日约十分钟可至筑基；其后金丹约一天、元婴两天、化神三天、后续各境约四天（纯挂机裸速，加成越高越快）——仙路虽长，日有所进。

## 离线收益

下线照修：再次上线按流逝时间结算修为与产出，**上限 48 小时**（轮回天赋可再延长）。道友们离线也各自修行、破境、炸炉、隐居，归来但见传闻满卷。

## 道友系统

本作风骨所在：每位道友有姓名、性格、流派、灵根与各自的命数。卷王进境神速，咸鱼得过且过，欧皇时来天地皆同力。你可以**论道**共进、**求助**解困、坊市**照顾生意**；好感六十，破境有挚友贺礼；好感一百，可结**道侣**——修炼速度永久 +10%，另有专属剧情传闻，红绸十里，仙鹤来贺。境界相仿者，亦可能化为你追我赶的**宿敌**，战书迭至。

## 轮回飞升

大乘十层圆满，可引动**九重天劫**：九波神雷逐波凶险，祭丹可壮行色，败则跌落大乘九层（修为不失，保底渐增）。渡尽九波即飞升转世：保留图鉴、成就与部分灵玉，携**轮回点**再入尘寰——十五节点天赋树（修炼/战斗/机缘三支）、十四种转世身份（书香门第、猎户之子、流浪儿……乃至万中无一的天弃之子），每一世皆是新局。

## 长线目标

- **图鉴**：功法、丹方、灵宠、装备、道友五类共约两百项（道友满编时 204 项），含自创功法。
- **镇妖塔**：无尽层数，每 33 层有隐藏 BOSS 守候。
- **成就**：八十枚，其中十枚隐而不显，达成方知。
- **轮回**：次数无上限，天赋树点满、身份图鉴集齐，漫漫长路。
- **道友**：满好感、结道侣、彩蛋名道友相逢即是缘。

## 机缘暗藏（不剧透）

炸炉未必是祸，子夜（凌晨 0~6 点）吐纳或有所遇，连锁奇遇步步生莲；丹房、塔顶、归墟、龙渊……皆有机缘，静候有缘人。

## 存档导出 / 导入

设置页 →「导出存档」复制字符串（base64），他处「导入存档」粘贴即可。存档仅存于本浏览器 localStorage（通常不足 100KB），清理浏览器数据前请先行导出。

## 目录结构（致开发者）

数据与逻辑严格分离，扩充内容一般只动 `js/data/`：

```
index.html      入口（按序加载全部脚本，无构建）
css/style.css   国风水墨样式
js/core/        引擎层：工具/事件/数值配置/状态/存档/属性聚合/离线结算
js/data/        纯数据表：功法/丹方/灵宠/装备/地图/事件/成就/道友文案（只登记，无逻辑）
js/sys/         玩法系统：修炼/功法/炼丹/道友……（各管一域，经 XG 命名空间协作）
js/ui/          界面层：tab 框架与各系统页
js/main.js      启动引导与一秒主循环
```

契约文档：`CONTRACTS.md`（接口唯一事实来源）、`DATA_NOTES.md`（数据层 id 速查）。
